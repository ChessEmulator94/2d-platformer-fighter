# 2D Platformer Fighting Game

This is a 2D platformer style street fighter esque game made using JS, HTML and CSS.

### How To Play

Player 1 controls:
W - Jump
A - Move Left
D - Move Right
Spacebar - Attack

Player 2 controls:
Up Arrow - Jump
Left Arrow - Move Left
Right Arrow - Move Right
Enter - Attack
